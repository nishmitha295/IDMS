**IDMS - Internal Data Management System**
IDMS (Internal Data Management System) is an integrated platform designed to streamline and digitize the internal operations of an organization. It manages data across departments such as HR, Store, Finance, Attendance, Memo, Reports, and more — all from a centralized dashboard.

Key Objectives:
Centralize internal departmental data

Improve transparency and collaboration between departments

Provide secure, role-based access control

Automate workflows like attendance, asset tracking, and memo issuance

**Modules Covered:**
**Attendance Management**

Track employee check-in/check-out

Manage leaves and working hours

**HR Management**

Handle employee onboarding/offboarding

Performance records and leave requests

Document and activity tracking

**Store & Asset Management**

Track physical assets: furniture, printers, etc.

Department-wise allocation and movement

**Finance Management**

Monitor fixed and variable expenses

Internet bills, salaries, water bills, SIM recharge

Incentives and other financial records

**Memo & Notices**

Send official communications internally

View past memos issued to departments or individuals

**Reports Module**

Generate internal performance, attendance, and financial reports

**Admin Panel**

Role-based access for Admin, HR, Finance, Store Manager, Employee

Dashboard overview with statistics and charts

**Why IDMS?**
Replaces manual paperwork with a digitized workflow

Centralized access to real-time internal data

Scalable architecture for multi-department collaboration

Enhances accountability and reduces overhead

**Built With:**
Frontend: Next.js (React + TypeScript)

Backend: Spring Boot (Java)

Database: MySQL

Authentication: JWT-based login system

Tools: VS Code, GitHub, Postman
